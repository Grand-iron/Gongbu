위 프로그램은 mysql을 활용하여 경찰청 분실물 및 대중교통 분실물 실시간 정보를 한 눈에 볼 수 있고, 검색 및 알림설정이 가능한 프로그램입니다.

*현재 완성프로그램.py에서 사용했던 서버는 대학교에서 제공한 서버이고, 시간의 경과에 따라 서버를 사용할 수 없습니다.*

대중교통 오픈 api는 인증키를 받기 위해서 사용 url을 적어야하기 때문에 이를 해결하지 못하여 csv파일로 대체했습니다.
알림설정은 프론트를 구성하지 않아 프로그램 내에서 트리거만 작동 되도록 만들어 놨습니다.

프로그램 작동 장면은 발표자료내 ppt 31p를 참고해주세요. 감사합니다.
